### 🔹 AI Music Generation Web App (Yumi) (LLM-Integrated MVP Platform) 
### (updated 05.13.2026)


🔗 [View Yumi Project on GitHub](https://github.com/lettersfromsubin/yumi-ai-music-generator)

🚀 [Live Demo on Hugging Face](https://huggingface.co/spaces/ALEXJK0901/yumi-ai-music-creator)

🎧 [SoundCloud Demo Tracks](https://soundcloud.com/oxl8s6qj4r1i?utm_source=clipboard&utm_medium=text&utm_campaign=social_sharing)

- Developed a Korean-first AI music generation web application that creates personalized music experiences from mood, genre preferences, lyrics, and creative textual input
- Built a React + Vite frontend with a guided multi-step music generation workflow for intuitive Human–AI creative interaction
- Integrated a hosted ACEMusic API pipeline through a secure Node.js backend architecture that prevents direct exposure of API credentials in the browser
- Implemented end-to-end music generation flow including user input processing, structured generation requests, generated audio playback, lyrics visualization, and album-cover preview generation
- Designed and deployed a Docker-based Hugging Face Space environment supporting server-side AI music generation workflows
- Implemented demo-safe deployment logic separating public demo playback from real API-powered generation
- Built configurable generation controls including mood, genre, vocal tone, duration, sound detail, and title-generation modes
- Explored Human–AI Interaction (HCI) and emotionally aware generative systems for AI-assisted creative expression

### 🔹 AI 음악 생성 웹 애플리케이션 'Yumi' (LLM 통합 MVP 플랫폼)
*(최종 업데이트: 2026.05.13)*

🔗 [GitHub에서 Yumi 프로젝트 보기](https://github.com/lettersfromsubin/yumi-ai-music-generator)  
🚀 [Hugging Face에서 Live Demo 체험하기](https://huggingface.co/spaces/ALEXJK0901/yumi-ai-music-creator)  
🎧 [SoundCloud 데모 트랙 듣기](https://soundcloud.com/oxl8s6qj4r1i?utm_source=clipboard&utm_medium=text&utm_campaign=social_sharing)  

- **개발 내용:** 사용자의 기분(Mood), 선호 장르, 가사 및 텍스트 프롬프트를 바탕으로 개인화된 음악을 생성하는 한국어 특화 AI 음악 생성 웹 애플리케이션 개발
- **프론트엔드:** React + Vite 기반으로 직관적인 인간-AI 상호작용(HCI)을 고려한 다단계(Multi-step) 음악 생성 워크플로우 설계
- **백엔드 및 보안:** 브라우저 내 API 키 노출을 방지하기 위해 Node.js 백엔드 아키텍처를 구축하고 ACEMusic API 파이프라인을 안전하게 통합
- **End-to-End 파이프라인:** 사용자 입력 처리, 구조화된 생성 요청, 오디오 재생, 가사 시각화 및 앨범 커버 프리뷰 생성까지 전체 음악 생성 흐름 구현
- **배포(Deployment):** Docker를 활용하여 Hugging Face Space 환경에 배포 및 서버사이드 AI 음악 생성 워크플로우 구축
- **데모 환경:** 실제 API 호출과 퍼블릭 데모 재생(Demo-safe) 로직을 분리하여 안정적인 서비스 환경 구현
- **기능 및 제어:** 기분, 장르, 보컬 톤, 곡 길이, 디테일 설정 및 자동 제목 생성 기능 등 사용자가 세밀하게 제어할 수 있는 생성 옵션 제공
- **연구적 의의:** AI 기반 창작 지원 시스템에서의 인간-AI 상호작용(HCI) 및 감정 인지형(Emotionally aware) 생성 시스템 탐구

*(기술 스택(Tech Stack) 및 세부 분야별 개발 내역은 상단의 영문 버전을 참고해 주시기 바랍니다.)*
